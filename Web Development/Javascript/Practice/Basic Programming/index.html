  1. Creation of JavaScript

    Before JavaScript, webpages were static, with only HTML loaded directly from the server and displayed in the browser. In 1995, the founder of Netscape decided to create a programming language for their browser. He hired Brendan Eich for this task, and Eich developed the first version of JavaScript in just 10 days. Initially, it was known as “Mocha”, which was later renamed to “LiveScript” and finally “JavaScript”. Due to conflicts with Microsoft, who developed their own version of JavaScript called JScript through reverse engineering, Netscape handed JavaScript over to ECMA. The Mozilla Foundation continued to develop JavaScript for the Firefox browser, marking the beginning of ES-1 or ECMAScript.

    Atwood's Law:

  2. Basics of JavaScript

    Javascript is an interpreter based language. An interpreter in the browser reads over the JavaScript code, interprets each line, and runs it. More modern browsers use a technology known as Just-In-Time (JIT) compilation, which compiles JavaScript to executable bytecode just as it is about to run. Chrome V8 engine is the interpreter for javascript and only because of this NodeJs is possible now.
